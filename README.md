# Seer SDK

Official SDK for developing Seer file preview plugins.

## Version

Current version: **v3.0** (2026-04-20)

## Requirements

- **Qt**: Qt 6.8
- **C++ Standard**: C++17
- **Compiler**: MSVC 2022
- **Build System**: CMake 3.22+
